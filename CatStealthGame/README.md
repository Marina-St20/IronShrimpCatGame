# Cat's Eye

## About The Game

This project is a 2D cat stealth game made during a 48-hour game jam hosted by UofU's GameCraft club. 

You play as a cat navigating the sewers. Win by avoiding rat patrols, collecting keys, and freeing the trapped cats. 

As a cat, stealthily navigate the dark, underground sewers the rats call home. With your beam eye ability, free your family and all the cats who were kidnapped by the rats, who wanted revenge on the cats for eating them. Stay out of sight, collect keys to rescue cats, and avoid hazards to win. 

## Play It

[Play on itch.io](https://marina-st20.itch.io/cats-eye)

## Features

- 2D stealth gameplay focused on avoiding patrols and hazards
- Keys and cages progressions with silver and gold keys
- A stamina system tied to the cat's eye beam mechanic
- Many scenes including the main leve, a starting screen, win screen, and lose screen

## Controls

- `WASD` or arrow keys to move
- Mouse for direction
- Click `F` to toggle beam eye lights

## Built With

- Unity 6 (`6000.4.0f1`) (C#)
- Universal Render Pipeline
- Unity Input System
- Unity 2D tools and lighting
